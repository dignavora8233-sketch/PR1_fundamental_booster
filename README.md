# Fundamental Booster

## Overview

This program is an interactive personal data collector.

## What it does

-   Welcomes the user.
-   Collects the user's name.
-   Collects the user's age.
-   Collects the user's height in meters.
-   Collects the user's favourite number.
-   Displays the collected information.
-   Shows the data type and memory address of each value.
-   Calculates an approximate birth year based on the entered age.
-   Displays a thank-you message before exiting.

## Inputs

-   Name
-   Age
-   Height (in meters)
-   Favourite number

## Outputs

-   User information summary
-   Data types of entered values
-   Memory addresses of stored values
-   Approximate birth year
-   Closing message
